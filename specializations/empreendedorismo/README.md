# Especialização: Empreendedorismo

## Objetivo
Aprender a estruturar, validar e operar um negócio com disciplina financeira.

## Trilhas sugeridas
1. Ideia → validação → MVP
2. Finanças do negócio (precificação, margem, caixa)
3. Vendas e go-to-market
4. Captação (equity/dívida) e noções de VC (opcional)

## Projetos
- Simular um negócio: projeção de caixa 12 meses
- Definir oferta + canais + métricas (1-pager)
