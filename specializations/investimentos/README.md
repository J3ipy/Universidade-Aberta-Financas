# Especialização: Investimentos

## Pré-requisitos
- Ter concluído pelo menos as etapas 1 e 2 do currículo base (finanças pessoais + fundamentos de investimentos).

## Objetivo
Sair do “vocabulário” e ir para a prática:
- entender produtos e riscos,
- construir e acompanhar uma carteira,
- fazer análises básicas (fatores, macro, valuation inicial).

## Trilhas sugeridas
1. Renda fixa (juros, duration, crédito)
2. Renda variável (ações, FIIs)
3. Fundos e ETFs
4. Gestão de risco e alocação
5. Noções de derivativos (opcional)

## Projetos (faça pelo menos 2)
- Planilha de alocação (com rebalanceamento)
- Diário de investimentos (hipóteses, decisões, lições)
- Estudo de caso: “por que um ativo caiu/subiu?” (sem operar; só análise)
