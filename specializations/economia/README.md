# Especialização: Economia

## Objetivo
Aprofundar fundamentos para:
- interpretar conjuntura (inflação, juros, atividade),
- entender políticas econômicas,
- conectar economia com mercado financeiro e negócios.

## Trilhas sugeridas
1. Microeconomia (consumidor, firma, estrutura de mercado)
2. Macroeconomia (renda, inflação, política monetária e fiscal)
3. Economia brasileira (história, instituições, ciclos)
4. Dados e econometria básica (opcional)

## Projetos
- Painel macro (indicadores + análise mensal)
- Resumo crítico de 1 paper/artigo por mês
