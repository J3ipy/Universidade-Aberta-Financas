# Extras

Conteúdos que ajudam muito, mas não são “obrigatórios” para seguir a grade.

## Educação do investidor (Brasil)

- Publicações e materiais do Portal do Investidor (CVM): guias, cadernos e livros.
- B3 Educação: cursos e trilhas para investidores.

## Economia e conjuntura

- Aulas abertas e seminários de universidades (ex.: IE/Unicamp).
- Relatórios e séries históricas (para praticar leitura de dados macro).

## Ferramentas úteis

- Planilhas: orçamento, juros compostos, amortização, alocação.
- Python/R: séries temporais, risco/retorno, backtests simples.
