# Ajuda

Se você ficou travado:

1. Procure no [FAQ](FAQ.md).
2. Abra uma issue com:
   - etapa/tema
   - onde você travou
   - o que já tentou
   - links/prints (sem dados pessoais)

Sugestões de estudo:
- faça projetos pequenos e frequentes
- revise pré-requisitos (estatística, matemática financeira, contabilidade)
- discuta dúvidas em comunidade (Discord/Telegram/etc. — se você criar um, coloque o link aqui)
